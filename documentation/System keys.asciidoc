
=== USBSTOR

....
HKEY_LOCAL_MACHINE\System\CurrentControlSet\Enum\USBSTOR
....

Sub key level 1: Disk&Ven_&Prod_&Rev_0.00
....
<Device Type>&Ven_<Vendor>&Prod_<Product>&Rev_<Revision Number>
....

Sub key level 2: 1002131402536a&0

*Sub keys:*

* Device Parameters
* LogConf
* Properties

*Values:*

* Capabilities
* Class
* ClassGUID
* CompatibleIDs
* ConfigFlags
* ContainerID
* DeviceDesc
* Driver
* FriendlyName
* HardwareID
* Mfg
* Service
